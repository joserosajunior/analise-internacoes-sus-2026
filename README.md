# Análise de Internações Hospitalares do SUS - Janeiro de 2026

## Objetivo

Este projeto tem como objetivo analisar dados públicos de internações hospitalares do SUS para identificar padrões relacionados ao perfil dos pacientes, custos hospitalares, permanência e mortalidade.

## Fonte dos Dados

* DATASUS
* Sistema de Informações Hospitalares (SIH)
* Período analisado: Janeiro de 2026

## Tamanho do Dataset

* 236.541 registros

## Perguntas de Negócio

1. Qual o perfil dos pacientes internados?
2. Quais diagnósticos geraram mais internações?
3. Quais diagnósticos geraram maior custo?
4. Existe relação entre idade e tempo de permanência?
5. Qual a taxa de mortalidade por diagnóstico?
